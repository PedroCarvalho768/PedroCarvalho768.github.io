Hello and welcome to my personal blog! This is my first post using the Jekyll Theme Console.

## About This Blog

This blog is built with:
- **Jekyll** - A simple, blog-aware static site generator
- **Jekyll Theme Console** - A minimalist, developer-focused theme with a terminal aesthetic
- **GitHub Pages** - For free hosting and easy deployment

## Why I Chose This Setup

I selected this theme because it offers:
- Clean, readable design perfect for technical content
- Four beautiful themes (dark, light, hacker, nord) with automatic OS preference detection
- Excellent performance as a static site
- Minimal dependencies and easy customization
- Strong security with built-in Content Security Policy

## What to Expect

In this blog, I plan to share:
- Programming tutorials and tips
- Technology insights and trends
- Personal projects and experiments
- Reflections on software development practices

Stay tuned for more content coming soon!
